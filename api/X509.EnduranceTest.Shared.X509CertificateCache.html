Class X509CertificateCache

Per-thread-cache of X509Certificate2 instances, identified by StoreName, StoreLocation, Thumbprint and an optional cache name. Use the cache ONLY IF you absolutely know how you are using the X509Certificate2 instance. Disposing the certificate, for example, will leave a stale and useless X509Certificate2 instance in the cache.
